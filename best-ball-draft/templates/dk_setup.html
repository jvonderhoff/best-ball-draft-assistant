🏈 DK Auth Setup

One-time setup so Best Ball Assistant can read your live draft from DraftKings.

Checking…

Open DraftKings in your phone browser (Safari or Chrome), log in, then follow these steps.

1Save a bookmark

Add any page as a bookmark in your browser, then edit the URL and replace it with this JavaScript:

javascript:(function(){var s=document.createElement('script');s.src='http://SERVER_IP:8000/static/dk-auth.js?t='+Date.now();document.head.appendChild(s)})();

Tap to copy · Replace SERVER_IP with your Mac's IP

2Open DraftKings and run the bookmark

Navigate to draftkings.com (any page, while logged in), then tap the bookmark you saved. It will send your session to Best Ball Assistant automatically.

3Come back here to verify

Return to this page — the status above should turn green. You're ready to draft.

Open DraftKings in your desktop browser, log in, then open the developer console (F12 → Console tab) and paste:

fetch('http://SERVER_IP:8000/api/dk-auth',{method:'POST',headers:{'Content-Type':'application/json'},body:JSON.stringify({cookie:document.cookie,storage:JSON.stringify(localStorage)})}).then(r=>r.json()).then(d=>console.log('BBA Auth:',d))

Tap/click to copy · Replace SERVER_IP with your Mac's LAN IP

Press Enter. You should see BBA Auth: {ok: true, …} in the console.

If the bookmarklet approach doesn't work, you can paste your DK cookie string here directly.

To get it: Open DK in a browser where you're logged in → F12 → Application → Cookies → https://www.draftkings.com → copy all cookie rows as a string like name=value; name2=value2
